Concepts Learnt in Yesterday Class

Topic Explanation
DNS Server A DNS server is a computer server that contains a database of public IP addresses and their associated hostnames, and in most cases serves to resolve, or translate, those names to IP addresses as requested.
Some Basic HTML Tags
  • Html Tag — The root element
  • Head Tag — The document head
  • Title Tag— The page title
  • Paragraph tag— A paragraph
  • Anchor Tag - Used for hyperlinks
  • Heading Tag - Provide different heading sizes
  • Image Tag - Input image in page
JSON

JSON stands for JavaScript Object Notation.

JSON supports only text and number data type.

JSON doesn't provide display capabilities.
XML

XML stands for eXtensible Markup Language..

XML support many data types such as text, number, images, charts, graphs etc. Moreover, XML offeres options for transferring the format or structure of the data with actual data.

XML provides the capability to display data because it is a markup language.
