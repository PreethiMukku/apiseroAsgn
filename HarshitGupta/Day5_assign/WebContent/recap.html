Day-5 Recap

HTML

		--HTML stands for HyperText Markup Language.
		--It is used to define the structure of a webpage.
		--There are many inbuilt tags in HTML each coming with its own usage and functionality
		  For Example table tag is used to layout a structure of a table.
		--There are attributes of each tag which adds extra features such as nowrap attribute
		  of a th tag.
		--Other HTML tag are p,div,heading,img

CSS

		--CSS stands for cascading stylesheet.
		--It is used to give styling to the html tags.
		--There are 3 ways to write css.
			--Inline css:here we write the styling as an atrribute of a tag within the style attribute.
			--Internal:Here we define the css in the head section within the style tag
			--Extertal:Here we use the link tag to link the external css file to this doc
		--External CSS is used so as it can be reused and is not tightly coupled
		--Examples of CSS: background-color,color,font-family

JS

		--Javascript is used for client side validation,DOM manupulation and asynchrous calls to the server
		--It is an interpreted language.
		--JS can also be used in 3 ways : Inline , Internal , External
		--It makes the UI more interactive
		--Some examples of Javascript methods are : getElementbyId(),querySelector()

XML vs JSON

		--XML stands for Extensible Markup Language and JSON stands for Javascript ObjectNotation
		--XML can have data with many datatypes whereas JSON has only string,int,boolean
		--JSON saves a lot of space as it is lightweight
		--XML can emmbed metadata along with the data whereas XML cant

Dynamic Web Pages

		--Server side sends the dynamic webpage according to the response received
		--It doesnot show the buisness logic as only the ouput file is sent.
		--It has same features as static pages of including css and js.
