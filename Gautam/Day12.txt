->The main reason for organizations to use integration is their need to improve productivity and quality of their operations. 

Security

JMS
->The Java Message Service (JMS) API is a messaging standard that allows application components based on the Java Platform Enterprise Edition (Java EE) to create, send, receive, and read messages.
->It enables distributed communication that is loosely coupled, reliable, and asynchronous.

Oauth
->OAuth is an open standard for access delegation.
->It iscommonly used as a way for Internet users to grant websites or applications access to their information on other websites but without giving them the passwords.
