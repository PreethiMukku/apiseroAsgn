DAY 5 Concepts at a Glance

Topics Description
D N S The Domain Name System (DNS) is the phonebook of the Internet. DNS translates domain names to IP addresses so browsers can load Internet resources.
HTML
  • HTML — The root element
  • Head — The document head
  • Title — The page title
  • Paragraph — A paragraph
  • Anchor - Used for hyper links
  • Heading - Provide different heading sizes
  • Image - Input image in page
JSON

JSON stands for JavaScript Object Notation.

JSON supports only text and number data type.

JSON doesn't provide display capabilities.
XML

XML stands for eXtensible Markup Language..

XML support many data types such as text, number, images, charts, graphs etc. Moreover, XML offers options for transferring the format or structure of the data with actual data.

XML provides the capability to display data because it is a markup language.
